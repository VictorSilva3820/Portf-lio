O Plana foi um projeto desenvolvido no fim de 2019, ele foi criado com o intuito de servir com uma agenda pessoal. Neste projeto eu estive responsável pela criação das funcionalidades de cadastro, as sessões, o calendário(que infelizmente não funcionou como queriamos) e a criação do banco de dados. Na criação do site foi utilizado HTML, CSS, PhP e JavaScript.

O Exact foi um site criado com intenção de auxilar estudantes do ensino médio, ele trás uma breve explicação de alguns assuntos da matemáticos e física, juntamente com um programa utilizado para mostrar o resultado de certas problemáticas. Ele foi desenvolvido utilizando HTML, CSS e inteiramente JavaScript. Nele eu desenvolvi o primeiro dos assuntos contidos no site, assim como fiz a criação do layout do site que foi feito em conjunto por todos os membros da equipe.

O Gastronomus foi um projeto destinado para o ramo de atividade alimenticia, onde o projeto seria um site criado para uma linha de restaurantes com o nome Gastronomus. Ele foi criado utilizando apenas HTML e CSS, e eu estive responsável pela criação da galeria( que funciona como um carrosel), e devido a alguns problemas acabei tendo de criar a parte voltada para contatos.
